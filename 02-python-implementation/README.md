# Python実装編

CIGRE TB820確率計画法の概念をPythonで実装する練習問題集です。

## 学習目標

- 確率計画法の数値計算手法
- Python最適化ライブラリの活用
- 大規模問題への対応
- 結果の可視化と分析

## 必要なライブラリ

```bash
pip install numpy scipy pandas matplotlib seaborn cvxpy pulp
```

## フォルダ構成

```
02-python-implementation/
├── README.md                   # このファイル
├── DOWNLOAD_LINK.md           # オンライン版リンク
├── src/                       # ソースコード
│   ├── __init__.py
│   ├── utils.py               # 共通関数
│   └── models/                # 最適化モデル
├── notebooks/                 # Jupyter Notebook
└── data/                     # サンプルデータ
```

## 実装内容

### 基本モジュール (src/)
- `utils.py`: 確率分布、統計計算、可視化関数
- `models/`: 各種最適化モデルの実装

### ノートブック (notebooks/)
- 対話的な学習とコード実行
- ステップバイステップの説明
- グラフとチャートによる結果可視化

### データ (data/)
- サンプル電力需要データ
- 気象データ（風速、日射量）
- 系統パラメータファイル

## 学習の進め方

1. **基礎編の理解**: まず理論問題を解く
2. **環境セットアップ**: 必要ライブラリのインストール
3. **サンプル実行**: 簡単な例から始める
4. **段階的発展**: 複雑な問題に挑戦
