# 問題2: リスク制約付き最適化

## 問題設定

送電線の熱容量制約を考慮したリスク管理問題。

### 与えられた条件

- **送電線の熱容量**: 100MW（これを超えると設備損傷のリスク）
- **負荷の不確実性**: 一様分布 U(80, 120)MW
- **発電機A**: 容量60MW、コスト50$/MWh
- **発電機B**: 容量80MW、コスト70$/MWh
- **制約違反の許容確率**: 最大5%

### 決定変数

- xA: 発電機Aの出力 [MW]
- xB: 発電機Bの出力 [MW]

## 問題

### (1) 確率制約の定式化

送電線容量制約違反の確率を5%以下に保つ制約を数式で表せ。

### (2) 期待運用コストの最小化

確率制約を満たしつつ、期待運用コストを最小化する発電計画を求めよ。

### (3) リスク指標の計算

最適解において以下を計算せよ：
- VaR（5%水準）
- CVaR（5%水準）

## 解答の方向性

### ステップ1: 確率制約の変換

負荷Lが一様分布U(80,120)に従う場合：
```
P(xA + xB < L) ≤ 0.05
```

これを確定的制約に変換する。

### ステップ2: 最適化問題の設定

```
minimize: 50xA + 70xB
subject to:
- 確率制約（変換後）
- 0 ≤ xA ≤ 60
- 0 ≤ xB ≤ 80
```

### ステップ3: リスク指標の計算

VaRとCVaRの定義に基づいて計算。

## 数学的詳細

### 一様分布の確率制約

負荷L ~ U(80, 120)の場合：
- P(L > 送電容量) = P(L > 100) = 0.5
- P(発電量 < L) ≤ 0.05 を満たす発電量の条件は？

### 分位点の活用

95%分位点を利用した制約：
```
xA + xB ≥ F⁻¹(0.95)
```

## 学習ポイント

1. **確率制約の実用性**: 100%安全 vs コストのトレードオフ
2. **分布の性質**: 一様分布の分位点計算
3. **リスク指標**: VaRとCVaRの関係と解釈

## 発展問題

- 制約違反コストが200$/MWh.hの場合、期待値アプローチとの比較
- 負荷分布が正規分布の場合の解法変更
- 複数の送電線制約がある場合の拡張
