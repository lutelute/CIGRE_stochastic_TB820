# 基礎編：理論的な練習問題

CIGRE TB820に基づいた確率計画法の基礎的な練習問題集です。

## 学習目標

- 確率計画法の基本概念の理解
- 不確実性のモデリング手法
- 期待値とリスク指標の計算
- 基本的な最適化問題の定式化

## 問題構成

### レベル1: 基本概念（問題1-3）
- 確率分布の理解
- 期待値計算
- 簡単な最適化問題

### レベル2: リスク評価（問題4-6）
- VaR (Value at Risk)
- CVaR (Conditional Value at Risk)
- 確率制約

### レベル3: 応用問題（問題7-10）
- シナリオ分析
- 二段階最適化
- 感度分析

## 問題一覧

1. [基本的な期待値最適化](problems/problem_01.md) - 風力発電の出力変動
2. [リスク制約付き計画](problems/problem_02.md) - 送電線容量制約
3. [シナリオベース最適化](problems/problem_03.md) - 需要不確実性
4. [VaRとCVaRの比較](problems/problem_04.md) - リスク指標の違い
5. [確率制約の定式化](problems/problem_05.md) - 停電リスクの管理

## 前提知識

- 線形代数
- 確率・統計
- 最適化の基礎
- 電力システムの基本概念

## 学習のヒント

1. **数学的記法に慣れる**: 確率変数、期待値、分散の記号
2. **段階的に理解**: 簡単な問題から始めて複雑さを増す
3. **図解を活用**: 確率分布や制約をグラフで可視化
4. **実際の値で計算**: 抽象的な式だけでなく具体的な数値例で確認
